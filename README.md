# What you get

- A hostname indicator in the top bar during screen sharing.

# Installation

- Download and move the *hostnameIndicator@chrisss404.gmail.com* directory to *~/.local/share/gnome-shell/extensions/*
- Reload GNOME Shell: log out and log back in
- Enable the extension: `gnome-extensions enable hostnameIndicator@chrisss404.gmail.com`
